# Switch Statements

Switch statements allow you to execute one block of code from multiple options based on the value of a variable.

## Syntax

```our_language
var grade = 'B';

switch (grade) {
    case 'A':
        print("Excellent");
        break;
    case 'B':
        print("Good");
        break;
    case 'C':
        print("Fair");
        break;
    default:
        print("Poor");
}
```

## Example

```our_language
var day = "Tuesday";

switch (day) {
    case "Monday":
        print("Start of the week");
        break;
    case "Friday":
        print("End of the work week");
        break;
    default:
        print("Midweek days");
}
```
